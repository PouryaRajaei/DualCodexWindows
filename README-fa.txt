Codex Dual Launcher
===================

این برنامه نصب بودن Codex و Codex Beta را برای کاربر فعلی ویندوز بررسی می‌کند.
اگر نسخه‌ای نصب نباشد، دکمهٔ صفحهٔ رسمی آن در Microsoft Store نمایش داده می‌شود.
پس از اجرا، فارسی‌ساز RTL نیز به‌صورت خودکار داخل رابط همان نسخه فعال می‌شود.

پروفایل‌ها:
Codex معمولی:
%USERPROFILE%\.codex\auth.json

Codex Beta:
%USERPROFILE%\.codex-beta\auth.json

اولین بار که هر نسخه را از داخل این لانچر اجرا می‌کنید، با حساب موردنظر وارد شوید.
از دفعات بعد، هر نسخه حساب مستقل خودش را حفظ می‌کند.

با دکمهٔ «تغییر پوشهٔ پروفایل» می‌توانید مسیر هر نسخه را جداگانه انتخاب کنید.
مسیرهای انتخاب‌شده برای دفعات بعد در تنظیمات لانچر ذخیره می‌شوند.

نکته:
برای اطمینان از اعمال CODEX_HOME، لانچر پردازش همان نسخه را قبل از اجرای مجدد می‌بندد.
نسخهٔ دیگر بسته نمی‌شود.
Codex معمولی از پورت محلی 9223 و Codex Beta از پورت محلی 9224 برای تزریق RTL
استفاده می‌کنند. این پورت‌ها فقط برای اتصال محلی DevTools هستند.

صفحه‌های رسمی Microsoft Store:
Codex: https://apps.microsoft.com/detail/9plm9xgg6vks
Codex Beta: https://apps.microsoft.com/detail/9n8cj4w95tbz
